# NOCTUA CORE SQUAD
Este squad contém a lógica base do framework de agentes AIOX.

## Responsabilidades
- **Intake:** Triagem e classificação de mensagens iniciais.
- **Memory:** Interface de persistência para sessões e dados.
- **Qualifier:** Qualificação de leads e coleta de dados cadastrais.
- **Buyer:** Lógica de interface de compra/fornecedores.

## Agentes
- `intake.md`: Definição de persona para triagem.
- `memory.md`: Definição de persona para gestão de histórico.
- `qualifier.md`: Definição de persona para coleta de dados.
- `buyer.md`: Definição de persona para interação com catálogos.
